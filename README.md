## SlotApiDemo
- Composable 함수를 다른 함수에 넣어주는 방법
- Composable 함수를 받은 Composable 은 런타임중에 동적으로 삽입이 되어, UI 를 구성할 수 있다
- 즉, Slot Api 를 이용한 Composable 은 본질적으로 런타임중에 삽입할 수 있는 하나 이상의 다른 Composable 을 포함하는 사용자 인터페이스 템플릿이다.
